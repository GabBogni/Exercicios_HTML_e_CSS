Tags de formatação

Certas tags criadas em versões antigas de HTML tem enfoque em efeito visual que estão ficando depreciadas

Bold vs Strong

A tag b (bold) tem função de negrito, mas não tem efeito semântico

A tag strong tem o mesmo efeito visual, mas tem efeito semântico

Italic vs Emphasis

Da mesma forma a tag i tem efeito visual enquanto a tag em tem um acrescimento de efeito semântico

Outros

Texto destacado com tag 'mark'

Texto com sentido de deletado com tag 'del'

Texto com sentido de preenchido com tag 'ins'

Texto sobrescrito com tag 'sup': X 2 e subscrito com tag 'sub': H2O

Texto pequeno dos contratos através da tag 'small'

Códigos

Através da tag 'code' é possível indicar que existe um código no documento. Para facilitar a leitura, a fonte se torna monoespaçada, então todos caracteres ocupam o mesmo espaço. Exemplo: let x = 2
Apesar disso é necessário a tag 'pre' para manter a formatação escrita no documento como quebra de linhas, espaços e tabulações 

 
                
                    let x = 2
                    x = 2 + 1
