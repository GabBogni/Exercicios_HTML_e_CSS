Imagens

Direitos autorais

É necessário se atentar aos direitos autorais das imagens. Exemplos de sites que possuem imagens de uso gratuito podem ser: Pexels, Unsplash

Formato de imagens

Os formatos de imagens mais comuns são png e jpg. O formato jpg é mais compacto do que png, porém possuem uma qualidade inferior, logo são indicadas para imagens grandes de plano de fundo. Além disso, imagens no formato png podem possuir transparência. Para diminuir o problema de imagens muito pesadas, grandes, um software de edição de imagens como GIMP pode ser utilizado
